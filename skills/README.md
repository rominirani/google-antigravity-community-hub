
# Skills
Skills are an open standard for extending agent capabilities. A skill is a folder containing a `SKILL.md` file with instructions that the agent can follow when working on specific tasks. 

Read the official docs on Skills [here](https://antigravity.google/docs/skills).


A list of recommended resources that demonstrate Antigravity Skills

| :--- | :--- |
| [Tutorial : Getting Started with Google Antigravity Skills](https://medium.com/google-cloud/tutorial-getting-started-with-antigravity-skills-864041811e0d) | This tutorial covers Antigravity Skills, the need for them, difference between Skills and MCP Server Tools/Antigravity Rules & Workflows, and practical examples that demonstrate multiple Skills that you can author and use in Antigravity. |
