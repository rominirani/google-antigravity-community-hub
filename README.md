# üåå Google Antigravity Community Hub

Welcome to the **Google Antigravity Community Hub**. This repository is the central "Mission Control" for developers learning to build with  [Google Antigravity](https://antigravity.google), an agentic-first development platform.

Here you will find a curated collection of codelabs, reusable skills, agent workflows, and community-driven resources to help you move from standard coding to orchestrating intelligent agents.

## üìÇ Repository Structure

We have organized resources into high-level categories to help you find exactly what you need.

| Folder | Description |
| :--- | :--- |
| **[`/getting-started`](./getting-started)** | Articles/Videos that are help to learn/get started with Antigravity. |
| **[`/codelabs`](./codelabs)** | Step-by-step tutorials that allow you to take Antigravity for a spin. |
| **[`/use-cases`](./use-cases)** | Articles/Videos that demonstrate building with Antigravity (e.g., Pomodoro Timers, Web Scrapers). |
| **[`/skills`](./skills)** | Extending Agent capabilities. Articles/Videos and Reusable capabilities (like Linters or API wrappers) you can drop into your project. |
| **[`/workflows`](./workflows)** | Orchestration patterns. Articles/Videos and Recipes for complex sequences (e.g., "TDD Cycle" or "Legacy Migration"). |
| **[`/rules`](./rules)** | Constraints and Context. Articles/Videos and Recipes that define coding standards, style guides, and boundaries for your Agents. |
| **[`/prompts`](./prompts)** | A library of effective system instructions and personas (e.g., "Senior Architect," "QA Tester"). |
| **[`/resources`](./resources)** | Any other reference material on Antigravity. |

## ü§ù Contributing

This is a community-driven project, and contributions are welcome! Whether you have created a new Skill, found a great Prompt, or built a cool Demo, please do share.

### How to contribute:
1. **Fork** this repository.
2. **Create** a new branch (`git checkout -b feature/amazing-new-skill`).
3. **Add** your content to the appropriate folder.
4. **Commit** your changes.

or share the content link and details in the [Issues](https://github.com/rominirani/google-antigravity-community-hub/issues/new).
